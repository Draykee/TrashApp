This app uses data from OpenStreetMap, so there's always a chance some trash cans aren't recognized yet.
If you find a new one, please take a moment to add it! :)


The quickest way to add things is by installing the OsmAnd app. If you already have it installed, you can tap the edit icon in the TrashApp Map view to launch it.


Make sure to enable the OSM Editing plugin via the sidebar!


There you can tap&hold at the position you want to add a trashcan, choose Actions and Create POI. In the POI Type field enter waste basket.
This will add the new trashcan for everyone to see within a couple of minutes.
